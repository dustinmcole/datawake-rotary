- **Description:** Weekly status summary for club leadership
- **Schedule:** Draft Saturday 8 PM PT, Send Monday 5 AM PT
- **Model:** claude-opus-4-6 (draft), claude-sonnet-4-6 (send)
- **Channel:** slack:#internal-rotary (draft review), email (final send)

## Two-Phase Process

### Phase 1: Draft (Saturday 8 PM PT)
1. Pull completed tasks from ClickUp (last 7 days)
2. Review upcoming events from Google Calendar (next 7 days)
3. Check for overdue items or blockers
4. Compile into update format
5. Post draft to #internal-rotary for review

### Phase 2: Send (Monday 5 AM PT)
1. Check if draft was approved (look for approval in Slack thread)
2. If approved (or approved with edits), send via email
3. If no response, hold and ping Dustin
4. If rejected, do not send
